# Doodle Jump

涂鸦跳跃风格的网页小游戏。操控像素风外星人向上跳跃，挑战更高分数。

## 快速开始

双击 `launch.bat` 即可在默认浏览器中启动游戏，或直接用浏览器打开 `index.html`。

**无需安装任何依赖，无需服务器，离线可玩。**

## 操作说明

### 移动

| 按键 | 动作 |
|------|------|
| `←` / `A` | 向左移动 |
| `→` / `D` | 向右移动 |

角色会自动在平台上弹跳，无需手动跳跃。从画面边缘出去会从对侧重新出现。

### 游戏状态切换

| 状态 | 按键 / 操作 | 结果 |
|------|------------|------|
| 开始界面 | `空格键` / 鼠标点击 | 开始游戏 |
| 游戏中 | `ESC` / 点击左上角 ⏸ | 暂停 |
| 暂停中 | `空格键` / 鼠标点击 | 继续游戏 |
| 暂停中 | `R` | 重新开始 |
| 游戏结束 | `空格键` / 鼠标点击 | 返回开始界面 |
| 游戏结束 | `R` | 直接重新开始 |

## 游戏机制

### 平台类型

| 颜色 | 类型 | 说明 |
|------|------|------|
| 绿色 | 普通平台 | 静止，可无限踩踏 |
| 蓝色 | 移动平台 | 左右来回移动 |
| 橙色 | 易碎平台 | 踩一次后消失 |
| 黄色 | 弹簧平台 | 弹力约 1.8×，直冲高空 |

### 道具

道具以 **10%** 概率浮现在平台正上方，碰触即拾取。

| 图标 | 类型 | 效果 | 持续时间 |
|------|------|------|---------|
| 橙红火箭 | 火箭 | 持续向上冲刺，无视平台碰撞，脚底喷火特效 | 约 3 秒 |
| 蓝色气泡 | 护盾 | ① 踩易碎平台不碎 ② 坠出底部自动弹回 | 约 8 秒 |

激活时左下角显示剩余秒数倒计时。

### 得分

- 角色上升高度越高，分数越高，分数只增不减
- 当前分数与历史最高分显示在右上角
- 每得 100 分有里程碑音效提示
- 历史最高分自动保存，刷新页面后仍然保留

### 难度递进

游戏难度随高度缓慢增加，共分 10 个区域（显示在右下角）：

- 移动平台和易碎平台的比例逐渐提升（普通 70%→35%，移动 20%→40%，易碎 10%→25%）
- 平台纵向间距逐渐拉大
- 移动平台速度逐渐加快

难度在区域 10 达到上限后保持不变。

## 技术说明

| 项目 | 内容 |
|------|------|
| 文件结构 | 单个 `index.html`，无外部依赖 |
| 渲染 | HTML5 Canvas API（420×680px） |
| 音效 | Web Audio API 纯合成，无音频文件 |
| 存储 | `localStorage` 保存最高分（Key: `doodleHighScore`）|
| 兼容性 | 支持所有现代桌面浏览器（Chrome / Firefox / Edge / Safari）|

## 代码结构

`index.html` 内的 `<script>` 按以下模块顺序组织：

1. **CONFIG** — 所有游戏常量（重力、跳跃力、平台尺寸、道具参数等）
2. **AudioManager** — Web Audio API 音效合成
3. **DifficultyManager** — 难度线性插值系统
4. **Platform** — 平台类（普通 / 移动 / 易碎 / 弹簧）
5. **Player** — 玩家类（物理 / 渲染 / 压缩拉伸动画）
5b. **PowerUp** — 道具类（火箭 / 护盾）
6. **GameState** — 全局状态变量
7. **Core Functions** — `init` / `update` / `render` / `gameLoop`
8. **Input** — 键盘与鼠标事件监听
9. **Bootstrap** — 启动入口
